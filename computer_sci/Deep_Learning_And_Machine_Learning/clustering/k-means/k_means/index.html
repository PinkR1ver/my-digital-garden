# Step by Step

Our algorithm works as follows, assuming we have inputsÂ $x_1, x_2, \cdots, x_n$Â and value of $K$

  • Step 1Â - Pick $K$ random points as cluster centers called centroids.
  • Step 2Â - Assign eachÂ $x_i$ to nearest cluster by calculating its distance to each centroid.
  • Step 3Â - Find new cluster center by taking the average of the assigned points.
  • Step 4Â - Repeat Step 2 and 3 until none of the cluster assignments change.

# To store the value of centroids when it updates
C_old = np.zeros(C.shape)
# Cluster Lables(0, 1, 2)
clusters = np.zeros(len(X))
# Error func. - Distance between new centroids and old centroids
error = dist(C, C_old, None)
# Loop will run till the error becomes zero
while error != 0:
    # Assigning each value to its closest cluster
    for i in range(len(X)):
        distances = dist(X[i], C)
        cluster = np.argmin(distances)
        clusters[i] = cluster
    # Storing the old centroid values
    C_old = deepcopy(C)
    # Finding the new centroids by taking the average value
    for i in range(k):
        points = [X[j] for j in range(len(X)) if clusters[j] == i]
        C[i] = np.mean(points, axis=0)
    error = dist(C, C_old, None)
